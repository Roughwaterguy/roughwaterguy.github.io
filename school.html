What are tags?

Html is written almost intirely in tags like this <html>,
and they end with this </html>, a closing tag
With this your page is structured in to different parts, kinda like a
essay or article. The page has three parts: the whole html
document, the head, and the body. All of which are vital for
a page to function. The <html> tag begins the document and
helps the browser read the page code. The <head> tag tells the
browser the background info on the page; such as the stylesheets,
javascript, and other things like links to data servers. Next you have
the <body> tag, its in this set of tags that the actual site
is layed out. Things like <h1>eadings and <p>aragraphs are
found.
